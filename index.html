개인정보 처리방침

스마트 히어 (SHPP) | 시행일: 2026년 2월 21일

1. 개요

스마트 히어(SHPP)는 사용자의 개인정보 보호를 최우선으로 생각합니다. 본 방침은 앱 사용 시 수집되는 정보와 그 처리 방법을 안내합니다.

2. 수집하는 정보

앱 사용 시 다음 정보가 기기 내에 저장됩니다:

  • 청력 테스트 결과 (주파수별 청력역치)
  • 개인 프로필 설정 (환경별 볼륨, 피팅 데이터)
  • 앱 사용 환경 설정

중요: 수집된 모든 데이터는 사용자 기기 내에만 저장되며, 외부 서버로 전송되지 않습니다.

3. 정보 이용 목적

  • 개인 맞춤형 청력 보조 서비스 제공
  • 환경별 음향 설정 최적화
  • Bluetooth 보청기 연동 기능 제공
  • 앱 기능 개선

4. 정보 공유 및 제3자 제공

스마트 히어는 사용자의 개인정보를 제3자에게 판매하거나 공유하지 않습니다.

5. 데이터 보안

모든 데이터는 사용자 기기 내 로컬 스토리지에 안전하게 저장됩니다. 인터넷 연결 없이도 완전한 기능 사용이 가능합니다.

6. 사용자 권리

  • 앱 내 설정에서 언제든지 데이터 삭제 가능
  • 앱 삭제 시 모든 데이터 완전 삭제

7. 어린이 개인정보 보호

본 앱은 만 14세 미만 아동을 대상으로 하지 않으며, 아동의 개인정보를 의도적으로 수집하지 않습니다.
